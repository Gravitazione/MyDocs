## About Frontend
ในส่วนของ Frontend จะมี Framework หลักๆที่เลือกใช้งานอย่าง React, Next, Astro
โดยที่หลักๆ จะเน้น React และ Next เป็นหลักในการพัฒนา Project ในส่วนของ Frontend
ไปจนถึงในส่วนของ Library หลักๆที่จะใช้งาน ยกตัวอย่างเช่น State Management ที่จะใช้ Zustand,
Jotai, เป็นหลัก โดยในส่วนของการพัฒนา UI ด้วย CSS Framework ก็จะใช้ Library อย่าง Tailwind CSS เป็นหลัก
หรือในส่วนของการใช้งาน UI Components มาปรับแต่งเพิ่มอย่างเช่น Shadcn UI, Aceternity UI, Next UI, Antd, etc.
และในส่วนอื่นๆของการพัฒนา ในส่วนของ Frontend สามารถอ่านเพิ่มเติมตามหัวข้อในส่วนต่างๆ
